# 📝 Markdown 与 HTML 特色功能指南 (Markdown & HTML Features Guide)

本指南旨在向您介绍本终端（owo_terminal）内置渲染器所支持的全部 Markdown 和 HTML 渲染功能。我们的渲染器是基于 `marked` 并进行深度定制开发的，支持数学公式、拓扑图、离线五线谱音乐播放器、警告提示框以及特定 HTML 元素交互。

---

## 1. 基础排版与列表 (Basic Syntax)
- **硬换行**：支持 `breaks: true`，在输入框中单次换行即会渲染为 HTML 换行，无需加双空格。
- **时髦列表**：支持 `smartLists` 智能缩进，在渲染复杂层级列表时具备更优雅的行距和点号排版。

---

## 2. 🌐 HTML 支持与安全防护 (HTML Support)
渲染器不仅兼容标准 Markdown，还对原生 HTML 和特定页面元素做了兼容和过滤优化：
- **行内 HTML 渲染**：支持在 Markdown 中直接混合书写标准的 HTML 标签进行定制排版。
- **特定组件嵌入**：支持解析 `<iframe>` 和 `<embed>` 标签，用于多媒体播放等高级渲染场景。
- **隐私保护与混淆**：
  - **邮箱混淆机制**：对识别到的邮箱自动使用 `htmlEncode` 随机编码为 HTML 字符实体（十六进制及十进制混合），并将 `@` 符号替换为 `#`。这可以彻底避免邮箱在 DOM 中被网络爬虫自动嗅探并收集隐私。
- **外链隔离防护**：所有生成的链接（包括 `markdown` 链接与特定 HTML 链接）均会被统一重写注入 `target="_blank"` 属性，强制在新窗口中开启，防止当前窗口被恶意网页篡改。

---

## 3. 🧮 数学公式 (LaTeX / KaTeX)
渲染器内置了 `katex` 引擎，支持直接嵌入 LaTeX 公式。
- **行内/行间公式**：使用双美元符号 `$$` 包裹。
  ```markdown
  $$e^{i\pi} + 1 = 0$$
  ```
- **特征**：完美支持数学符号与矩阵渲染。

---

## 4. 📊 拓扑图与流程图 (Mermaid)
可以直接使用 `mermaid` 语言渲染图表。
- **语法**：将代码块的语言标记设定为 `mermaid`。
  ```markdown
  ```mermaid
  graph TD
      A[开始] --> B(处理)
      B --> C{是否成功}
      C -- 是 --> D[结束]
      C -- 否 --> E[重试]
      E --> B
  ```
  ```
- **核心逻辑**：采用前端防抖和防重复重绘渲染，主题变更时会自动重新渲染适配亮色与暗色模式。

---

## 5. 🎵 五线谱记谱与离线播放 (ABCjs)
支持通过标准的 `abc` 记谱法渲染五线谱，并支持离线音频合成与高亮播放。
- **语法**：将代码块的语言标记设定为 `abc`。
  ```markdown
  ```abc
  X: 1
  T: 欢乐颂 (Ode to Joy)
  M: 4/4
  L: 1/4
  K: G
  |: B B C D | D C B A | G G A B | B3/2 A/2 A2 :|
  ```
  ```
- **播放器控制**：包含「播放」、「暂停」、「重新开始」与「进度条」的精致控制栏。
- **100% 纯本地离线**：系统内置 2.5MB 本地 Soundfont（位于 `/statics/soundfont/`），无须网络即可发声。
- **音符同步高亮**：播放时，五线谱上的音符会跟随音乐播放进度自动高亮变粉/红色，并伴随滤镜自适应主题暗色化。

---

## 6. 💡 警告提示框 (Admonition)
扩展了标准的警告提示框块语法，以便于在 Markdown 中展示带有视觉效果的卡片。
- **语法**：使用 `!!! 类型 "标题"` 的形式包裹。
  ```markdown
  !!! note "注意"
      这是一条普通的注意信息。
  ```
- **支持的类型**：
  - `note` / `abstract` / `info`
  - `tip` / `success` / `question`
  - `warning` / `failure` / `danger`
  - `bug` / `example` / `quote`
  - `hint` / `caution` / `error` / `attention`
- **特征**：不同类型自动适配不同的专属图标、边框主色调与淡雅背景。

---

## 7. 📁 智能附件解析 (Attachment)
当在输入框中粘贴截图或拖入文件时，系统会自动生成附件标识。
- **语法**：`[attachid:文件ID]`。
- **渲染特征**：
  - **图片附件**：渲染为最大宽高 `15rem`、带有精致圆角阴影、点击直接缩放放大预览的图片卡片。
  - **非图片附件**：渲染为精美的链接卡片，带有文件图标 📁 与一侧装饰边框。

---

## 8. 🔒 隐藏内容 (Hide BBCode)
支持论坛式的隐藏内容标签，用于在帖子中收纳或控制内容可见度。
- **语法**：
  - `[hide]隐藏内容[/hide]`
  - `【本内容已隐藏，回复后刷新可见哦】`
- **样式**：渲染为带有精细虚线边框和专属状态色的小表格卡片。

---

## 9. 📺 多媒体原生嵌入
支持一些特定的音乐和视频解析语法：
- **Bilibili**：`[bilibili]aid-bvid-cid[/bilibili]`
- **QQ 音乐**：`[qqmusic]songid-songtype[/qqmusic]`
- **Flash 嵌入**：`[flash]url[/flash]` 或 `[flash=宽,高]url[/flash]`
- **特征**：自动生成宽高度自适应的多媒体 `iframe` 播放面板。
